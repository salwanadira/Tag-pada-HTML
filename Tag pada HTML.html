Tag pada HTML beserta Kegunaannya


No. Nama Tag Deskripsi
1. < !DOCTYPE > Untuk menentukan tipe dokumen
2. < html > Untuk membuat sebuah dokumen HTML
3. < title > Untuk membuat judul dari sebuah halaman
4. < body > Untuk membuat tubuh dari sebuah halaman
5. < h1 > to < h6 > Untuk membuat heading (ukuran huruf)
6. < p > Untuk membuat paragraf
7. < br > Untuk memasukkan satu baris putus
8. < hr > Untuk membuat perubahan dasar kata didalam isi
9. < !--...-- > Untuk membuat komentarf
10. < a > Untuk membuat link
11. < b > Untuk membuat huruf bercetak tebal
12. < wbr > Untuk membuat kemungkinan garis putus-putus
13. < del > Untuk membuat teks yang telah dihapus dari dokumen
14. < var > Untuk membuat sebuah variabel
15. < abbr > Untuk membuat sebuah singkatan
16. < form > Untuk membuat sebuah form HTML untuk input pengguna
17. < input > Untuk membuat sebuah kontrol input
18. < blockquote > Untuk membuat sebuah bagian tezt yang dikutip dari sumber lain
19. < cite > Untuk membuat judul karya
20. < code > Untuk membuat potongan kode komputer di antara text
21. < dfn > Untuk membuat istilah definisi
22. < ins > Untuk membuat teks yang telah dimasukkan ke dalam dokumen
23. < kbd > Untuk membuat input keyboard
24. < mark > Untuk membuat teks yang disorot / ditandai
25. < pre > Untuk membuat teks terformat
26. < meter > Untuk membuat pengukuran skalar
27. < q > Untuk membuat kutipan pendek
28. < progress > Untuk memperlihatkan kemajuan tugas
29. < s > Untuk membuat teks yang tidak lagi benar
30. < samp > Untuk membuat contoh keluaran dari program komputer
31. < small > Untuk membuat teks kecil
32. < u > Untuk membuat teks yang memiliki gaya yang berbeda dari teks biasa lainnya
33. < strong > Untuk membuat teks penting
34. < sub > Untuk membuat teks subskrip (seperti dalam penulisan Zat Kimia)
35. < sup > Untuk membuat teks superscripted (seperti dalam penulisan akar kuadrat)
36. < time > Untuk membuat tanggal/waktu
37. < textarea > Untuk membuat sebuah kontrol input multibaris (text area)
38. < button > Untuk membuat sebuah tombol yang dapat diklik
39. < select > Untuk membuat sebuah daftar drop-down
40. < label > Untuk membuat sebuah label untuk sebuah elemen < input >
41. < datalist > Untuk menentukan daftar pilihan yang telah ditetapkan untuk kontrol input
42. < output > Untuk membuat hasil penghitungan
43. < iframe > Untuk membuat sebuah bingkai
44. < img > Untuk membuat gambar
45. < map > Untuk membuat gambar peta
46. < area > Untuk membuat area dalam gambar peta
47. < canvas > Untuk menggambar grafik, melalui scripting (JavaScript)
48. < figcaption > Untuk membuat sebuah caption untuk elemen < figure >
49. < figure > Untuk menentukan konten mandiri
50. < audio > Untuk membuat isi suara
51. < source > Untuk membuat sumber beberapa media untuk elemen media ( < video > dan < audio > )
52. < track > Untuk membuat trek teks untuk elemen media ( < video > dan < audio > )
53. < video > Untuk membuat sebuah video atau film
54. < nav > Untuk membuat navigasi link
55. < ul > Untuk membuat daftar dengan selain nomor
56. < ol > Untuk membuat daftar nomor
57. < li > Untuk membuat sebuah item daftar
58. < dl > Untuk membuat sebuah daftar definisir
59. < dt > Untuk membuat istilah (item) dalam daftar definisi
60. < menu > Untuk membuat deskripsi dari item dalam daftar definisi
61. < table > Untuk membuat tabel
62. < th > Untuk membuat sebuah sel header tabel
63. < tr > Untuk membuat baris dalam sebuah tabel
64. < td > Untuk membuat sel dalam sebuah tabel
65. < thead > Untuk mengelompokkan isi header dalam sebuah tabel
66. < tbody > Untuk mengelompokkan isi tubuh dalam sebuah tabel
67. < tfoot > Untuk mengelompokkan isi footer dalam sebuah tabel
68. < header > Untuk membuat sebuah header untuk dokumen atau bagian
69. < footer > Untuk membuat footer untuk dokumen atau bagian
70. < section > Untuk membuat bagian dalam dokumen
71. < hgroup > Untuk pengelompokkan elemen heading ( < h1 > sampai < h6 > )
72. < article > Untuk membuat sebuah artikel
73. < style > Untuk membuat informasi style untuk dokumen
74. < aside > Untuk membuat konten lain selain dari konten halaman
75. < details > Untuk membuat rincian tambahan yang pengguna dapat lihat / sembunyikan
76. < dialog > Untuk membuat sebuah kotak dialog atau jendela
77. < summary > Untuk membuat sebuah judul terlihat untuk elemen < detil >
78. < head > Untuk membuat informasi tentang dokumen
79. < meta > Untuk membuat metadata tentang dokumen HTML
80. < base > Untuk menentukan URL dasar / target untuk semua URL relatif dalam dokumen
81. < script > Untuk membuat script di sisi klien
82. < noscript > Untuk membuat sebuah konten alternatif bagi pengguna yang tidak mendukung script di sisi klien
83. < embed > Untuk membuat sebuah wadah untuk aplikasi eksternal (non-HTML)
84. < object > Untuk membuat sebuah objek yang ditanam
85. < param > Untuk membuat sebuah parameter untuk objek
